# 실습 체크리스트

## 사전 준비
### 코어 개발 실습 환경 셋업
- [ ] `go` 설치
- [ ] `IDE` 설치
- [ ] `git clone git@github.com:zenit9hub/byd50-ssi.git`
- [ ] `.env`에 `ETH_PRIVATE_KEY_HEX` 설정
- [ ] `make` 설치

### 안드로이드 데모앱 실습 환경 셋업
- [ ] `Android Studio` 설치
- [ ] `Android Studio`에 AVD emulator 설치
- [ ] `Android Studio`에 CMake, NDK 설치
- [ ] `make android` c-shared library 생성 및 JNI 연동부 코드 리뷰
